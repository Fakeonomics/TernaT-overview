Abstract

Neural reasoning at
microcontroller scale.

Combining vector-symbolic architectures with learned ternary neural networks for first-of-its-kind embedded reasoning.

We present TernaT, the first learned neural reasoner over Vector-Symbolic Architecture (VSA) using discrete ternary weights {−1, 0, +1}. Our system combines a 16 KB ternary TernMLResonator, a <1 KB FastController, and a <1 KB ChainScorer to achieve 90% exact multi-hop QA on a synthetic benchmark of 96 facts and 30 queries — without any LLM, GPU, or floating-point hardware.

The complete pipeline fits in <100 KB, deployable on $0.50 microcontrollers. Key insight: VSA superposition noise (~80% ceiling) is resolved via a hybrid exact+VSA memory architecture, while negative training eliminates false positives in the learned resonator.

About

Independent research.
Proprietary technology.

TernaT is an independent research project at the intersection of vector-symbolic architectures and ternary neural networks.

TernaT demonstrates that learned neural reasoning over Vector-Symbolic Architectures is not only possible but practical — achieving 90% exact multi-hop QA with a <100 KB pipeline deployable on $0.50 microcontrollers. The key insight: a hybrid exact+VSA memory architecture resolves the ~80% superposition noise ceiling inherent in pure VSA approaches, while negative training eliminates false positives in the learned resonator.

Combined with TernML for perception (38 KB), TernaT forms the backbone of a complete on-device AI pipeline that requires no LLM, no GPU, and no floating-point hardware.
